## üì¶ Installation

1. Clone the repository:
```bash
git clone https://github.com/AronnoSinghDurjoy/aronnosinghdurjoy.github.io.git
cd aronnosinghdurjoy.github.io
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## üåê Deployment to GitHub Pages

### Initial Setup

1. Make sure you have created a repository named `aronnosinghdurjoy.github.io` on GitHub

2. Update your GitHub repository URL in package.json (already configured)

### Deploy Steps

1. Build and deploy:
```bash
npm run deploy
```

This command will:
- Build your production app
- Create a `gh-pages` branch
- Push the build to GitHub Pages

2. Your site will be live at: **https://aronnosinghdurjoy.github.io**

### Manual Deployment (Alternative)

```bash
# Build the project
npm run build

# The build folder contains your production-ready files
# Deploy the build folder to your GitHub Pages
```

## üé® Customization

### Update Personal Information

1. **Hero Section** (`src/components/Hero/Hero.js`):
   - Change name and typing animations
   - Update social media links

2. **About Section** (`src/components/About/About.js`):
   - Modify about text
   - Update statistics

3. **Skills Section** (`src/components/Skills/Skills.js`):
   - Add/remove skills
   - Adjust skill levels

4. **Projects Section** (`src/components/Projects/Projects.js`):
   - Add your projects
   - Update project details and links

5. **Experience Section** (`src/components/Experience/Experience.js`):
   - Add your work experience
   - Update job details

6. **Contact Section** (`src/components/Contact/Contact.js`):
   - Update contact information
   - Configure form submission (add backend API)

### Change Colors

Edit CSS variables in `src/index.css`:

```css
:root {
  --primary-color: #6366f1;
  --secondary-color: #8b5cf6;
  --accent-color: #ec4899;
  /* ... more colors */
}
```
